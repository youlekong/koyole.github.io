分别使用gulp和webpack实践javascript source map

1.什么是source map

source map是一个信息文件，里面储存着位置信息。压缩后的代码执行过程中发生错误时，可以通过source map定位错误位置，对线上的项目排错十分有用。

2.生成source map

分别使用gulp和webpack生成javascript source map

1.gulp

  • 通过node脚本添加依赖包
  • gulp配置文件
  • 执行脚本命令

2.webpack（version: >= 4.0）

  • 通过node脚本添加依赖包
  • webpack配置文件
  • package.json中添加执行脚本
  • 执行脚本命令
